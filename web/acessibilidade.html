Acessibilidade

O Protege.Dados foi construído para navegação por teclado, leitura por tecnologias assistivas, contraste adequado e uso responsivo.

Personalize sua experiência

Recursos implementados

Acesso inclusivo

Navegação por teclado

Ordem lógica de foco, link para pular ao conteúdo e controles acessíveis.

Leitores de tela

HTML semântico, idioma declarado e mensagens dinâmicas com regiões de status.

Contraste e foco

Estados de foco visíveis e opção de alto contraste persistente.

Tipografia legível

Texto escalável, espaçamento confortável e opção de ampliação.

Movimento reduzido

Preferências do sistema são respeitadas para reduzir animações.

Sem dependências externas

O portal funciona com HTML, CSS e JavaScript locais no GitHub Pages.

Compromisso

O portal adota como referência as WCAG 2.2 e o eMAG, com evolução contínua baseada em testes e feedback.
