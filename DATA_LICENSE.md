# Licenciamento dos dados

O código deste repositório utiliza licença MIT. Os dados derivados são disponibilizados para reúso com atribuição da iniciativa e preservação das referências às fontes oficiais, respeitando as condições de uso dos órgãos de origem.

Citação sugerida:

> CARVALHEIRO, Ana Maria. Protege.Dados: Observatório de Proteção Digital Infantojuvenil. Brasília, 2026.

Os arquivos brutos permanecem sujeitos às condições e licenças das respectivas fontes oficiais.
