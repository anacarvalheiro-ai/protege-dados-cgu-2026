# Protege.Dados 4.0 — Release Final de Submissão

Este é o pacote congelado para submissão. Ele preserva a base territorial já existente no repositório e atualiza apresentação, resultados atuais, links, QR Codes, API, qualidade e automação.

## Instalação
1. Coloque o ZIP em Downloads.
2. Extraia-o para `Downloads\Protege_Dados_4_0_Final_Submissao`.
3. No PowerShell:
```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
& "$HOME\Downloads\Protege_Dados_4_0_Final_Submissao\scripts\Atualizar_Protege_Dados_4_0_FINAL.ps1"
```
4. Digite somente `PUBLICAR` quando solicitado.

## Verificação
Após o GitHub Actions ficar verde, teste o portal em janela anônima e siga `docs/4.0/PENTE_FINO_FINAL_SUBMISSAO.md`.
