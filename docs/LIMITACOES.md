# Limitações

- A IVPD é experimental.
- Não constitui ranking.
- Não mede prevalência.
- Não identifica pessoas.
- Não realiza previsão individual.
- Pode haver subnotificação.
- As fontes possuem períodos distintos.
- O produto não substitui diagnósticos locais.
- O eixo de capacidade territorial de proteção ainda não foi publicado.
