# Acessibilidade

O portal incorpora:
- link para pular ao conteúdo;
- navegação por teclado;
- foco visível;
- contraste institucional;
- HTML semântico;
- textos associados aos valores dos gráficos;
- design responsivo;
- linguagem de interpretação responsável.

A conformidade completa com WCAG 2.2 AA e eMAG deverá ser registrada em relatório de verificação manual e automatizada após a publicação pública.
