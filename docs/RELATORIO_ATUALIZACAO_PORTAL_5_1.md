# Relatório de atualização do portal — Protege.Dados 5.1

A interface foi reconstruída com base nas telas de referência aprovadas, mantendo o projeto autocontido e compatível com GitHub Pages.

## Atualizações

- identidade visual unificada;
- cabeçalho e rodapé azuis sem imagens decorativas;
- painel nacional com totais carregados da base JSON;
- cartograma funcional das 27 UFs;
- comparação de até três territórios;
- exportações CSV preservadas;
- dados abertos com filtros por texto, categoria, formato e escopo;
- metodologia, governança e acessibilidade redesenhadas;
- menu móvel e preferências persistentes de acessibilidade;
- scripts de atualização, prévia e publicação segura.
