# Metodologia do IVPD

O IVPD experimental combina indicadores territoriais de denúncias e conectividade escolar. A versão atual utiliza:

- 50% para a taxa normalizada de denúncias;
- 25% para o déficit normalizado de internet escolar;
- 25% para o déficit normalizado de banda larga escolar.

A normalização utiliza limites robustos com percentis 5 e 95. O resultado final é convertido para escala de 0 a 100.

## Limitações

- registros administrativos não equivalem à prevalência real;
- conectividade declarada não garante qualidade;
- comparações exigem o mesmo período, fonte e versão;
- o índice não deve fundamentar decisões automáticas.
