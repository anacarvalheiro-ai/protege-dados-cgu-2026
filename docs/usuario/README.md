# Guia do usuário

1. Selecione uma Unidade da Federação.
2. Leia os cartões de indicadores.
3. Consulte a explicação de cada número.
4. Compare UFs somente com o mesmo período e versão.
5. Use os downloads para análise complementar.
6. Consulte metodologia e limitações antes de divulgar conclusões.

O IVPD não representa percentual de crianças em risco.
