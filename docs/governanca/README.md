# Governança

## Controles

- versionamento semântico;
- changelog;
- testes automatizados;
- backup antes de atualizações;
- rastreabilidade de fontes;
- revisão humana;
- documentação das limitações;
- ausência de identificação individual.

## Gestão de mudanças

Toda alteração em fonte, peso, normalização ou regra de cálculo deve:

1. receber nova versão;
2. ser registrada no changelog;
3. executar os testes;
4. atualizar a metodologia;
5. informar impactos na comparabilidade.
