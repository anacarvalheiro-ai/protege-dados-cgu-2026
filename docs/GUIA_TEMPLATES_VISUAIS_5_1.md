# Guia dos templates visuais — Protege.Dados 5.1

A versão 5.1 consolida sete templates responsivos e autocontidos:

1. página inicial e painel nacional;
2. painel por UF e cartograma territorial;
3. comparador entre até três UFs;
4. dados abertos, downloads e API;
5. metodologia do IVPD;
6. governança e salvaguardas;
7. acessibilidade.

## Padrão visual

- cabeçalho e rodapé em gradiente azul institucional;
- ausência de imagens decorativas no cabeçalho e rodapé;
- tipografia de alta legibilidade;
- cartões brancos com bordas suaves e sombra discreta;
- navegação ativa destacada em azul;
- foco de teclado visível;
- layout fluido entre 320 px e telas amplas;
- tabelas com rolagem interna em telas estreitas;
- componentes e dados sem dependências externas.

## Funcionalidade

Os templates não são imagens estáticas. Todos os elementos foram implementados em HTML, CSS e JavaScript, preservando consulta, comparação, filtros, downloads, API, acessibilidade e publicação pelo GitHub Pages.
