## Escopo da versão

Esta versão consolida o Protege.Dados para apresentação institucional, demonstrações, reúso responsável de dados, submissões a editais e cooperação técnica.

## Principais entregas

- portal público responsivo;
- consulta por Unidade da Federação;
- indicadores territoriais;
- IVPD experimental;
- comparador entre UFs;
- arquivos CSV e JSON;
- API estática;
- metodologia documentada;
- governança e salvaguardas;
- canal de feedback por GitHub Issue, e-mail e WhatsApp;
- documentação para usuários e desenvolvedores.

## Interpretação responsável

O IVPD:

- não representa percentual de crianças em risco;
- não mede pessoas individualmente;
- não estima prevalência real da violência;
- não constitui ranking absoluto;
- não realiza decisões automatizadas;
- não substitui diagnóstico local ou avaliação humana.
