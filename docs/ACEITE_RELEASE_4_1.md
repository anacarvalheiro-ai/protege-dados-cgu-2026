# Critérios de aceite — Release 5.1

Este arquivo mantém o nome anterior por compatibilidade.

A versão 5.1 é aceita quando:

- os validadores da release e do portal são aprovados;
- os 3 testes Python são aprovados;
- `site.js` e `app.js` passam em `node --check`;
- a auditoria funcional confirma navegação, filtros, comparação, downloads, acessibilidade e responsividade;
- o portal contém exatamente 27 UFs e os totais publicados correspondem às bases derivadas;
- não há links locais ausentes nem erros de console;
- o cabeçalho e o rodapé mantêm a identidade azul institucional sem imagens decorativas.
