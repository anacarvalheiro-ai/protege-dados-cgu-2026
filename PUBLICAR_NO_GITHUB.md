# Publicação do Protege.Dados 5.1 no GitHub

## Fluxo recomendado

1. Execute `ATUALIZAR_PROTEGE_DADOS_5_1.bat`.
2. Revise a prévia com `ABRIR_PREVIA_LOCAL.bat`.
3. Confirme os relatórios em `evidence/audit/`.
4. Execute `PUBLICAR_NO_GITHUB_5_1.bat` dentro do repositório atualizado.
5. Digite `PUBLICAR` somente após revisar as alterações mostradas por `git status`.
6. Acompanhe os workflows **Qualidade Protege.Dados 5.1** e **Publicar Protege.Dados 5.1**.

A publicação não é automática: o script exige confirmação explícita antes do commit e do push.
